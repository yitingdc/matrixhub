# 用户管理

只有平台管理员才能访问和操作用户管理功能。

## 前置条件

- 已使用平台管理员账号登录 MatrixHub。

## 用户列表概览

用户管理页面包含以下信息和操作：

| 参数 | 描述 |
|-----------|-------------|
| 用户名 | 用户账号的唯一标识符。 |
| 平台管理员 | 指示该用户是否具有平台管理员权限。 |
| 来源 | 用户的来源，支持 **内置** (Built-in) 或 **LDAP**。 |
| 创建时间 | 用户账号的创建时间。 |
| 操作 | 支持 **设为管理员**、**取消管理员**、**重置密码** 和 **删除**。 |

![用户概览](./images/user-overview.png)

## 创建用户

1. 导航到 **平台设置** -> **用户管理** 页面，点击 **创建用户**。
1. 输入用户名、密码并确认密码；如果需要，勾选 **设为管理员**。
1. 点击 **确认** 完成创建。

![创建用户](./images/user-create.jpg)

## 重置密码

1. 在用户列表中找到目标用户，点击 **重置密码**。
1. 输入新密码并确认密码。
1. 点击 **确认** 提交。

![重置密码](./images/user-reset-password.jpg)
